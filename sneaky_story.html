│                                SNEAKY STORY                                  │
│                                                                              │
│                                                                              │
|     SneakyStory is a chrome extension that allows you to not                 |
|     appear in the "Who Viewed My Instagram Story" list                       |
|     of everyone's Instagram stories.                                         |
|                                                                              |
|     This extension only works on chrome based web browsers.                  |
|                                                                              |
|     Tested on:                                                               |
|     - Brave 1.20.108 (Chromium: 88.0.4324.182);                              |
|     - Brave 1.24.84 (Chromium: 90.0.4430.93);                                |
|                                                                              |
|     Installation:                                                            |
|     - [ Download the folder with the files ]                                 |
|     - Go to chrome://extensions/                                             |
|     - Enable "Developer mode"                                                |
|     - Click on "Load Unpacked" and select the downloaded folder              |
|                                                                              |
|     Usage:                                                                   |
|     - After the installation just click on the                               |
|       extension's icon to activate it.                                       |
|     - If the icon turns green: the blocker feature is activated              |
|       and nobody will see your account in their list                         |
|       if you see their story.                                                |
|     - To disable the blocker just click on the extension's icon.             |
|                                                                              |
|     Each time you open the browser enable and disable the blocker            |
|     a few times to reset the status and to allow the extension               |
|     to work properly.                                                        |
|                                                                              |
|                                                                              |
└──────────────────────────────────────────────────────────────────────────────┘
DOESN'T WORK ON MANIFEST V3


I will not port the extension to other browsers nor upload it to the chrome web store.
